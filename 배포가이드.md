# REELS LAB — 배포 가이드

## 1단계: GitHub에 올리기

1. [github.com](https://github.com) 가입 (이미 있으면 로그인)
2. 오른쪽 상단 `+` → `New repository`
3. Repository name: `reels-lab`
4. `Create repository` 클릭
5. 이 폴더 파일들을 모두 업로드 (Upload files 클릭)

---

## 2단계: Vercel에 배포

1. [vercel.com](https://vercel.com) → GitHub 계정으로 로그인
2. `Add New Project` → `reels-lab` 선택
3. 설정 변경 없이 `Deploy` 클릭
4. 1-2분 후 `https://reels-lab-xxxx.vercel.app` 링크 생성됨 ✅

이 링크를 팀원에게 공유하면 바로 사용 가능!

---

## 3단계: RapidAPI 키 발급

1. [rapidapi.com](https://rapidapi.com) 가입
2. 검색창에 `tiktok-scraper7` 검색
3. `TikTok Scraper` (tikwm) 선택 → Subscribe to Test
4. Basic 플랜 (무료: 월 500회 요청)
5. `X-RapidAPI-Key` 복사
6. 앱 우상단 ⚙️ 설정 → API 키 붙여넣기
